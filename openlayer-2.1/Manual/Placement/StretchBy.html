StretchBy

void StretchBy(
float factor )

Multiplies the stretching value of the Position by the specified amount.

Examples

Placement myPlacement( Vec2D( 250.0, 100.0 ), 0.5 * AL_PI );

// Multiply the stretching value of the Position by 1.5 //
myPlacement.StretchBy( 1.5 );
